# CaptureBrainRots

CaptureBrainRots is a Roblox game that combines **creature collection, combat, and tycoon** mechanics. The primary loop involves hunting down "Brain Rot" animals in the wild, capturing them, and using them to generate passive income in your base.

## Core Mechanics & Features

* **Hunting & Capture:** Use weapons to fight wild "Brain Rot" animals. Reduce their health to capture them.
* **Tycoon/Base Building:** Place captured animals on specific pads within your personal base to build out your tycoon.
* **Passive Income:** Animals placed on pads generate a steady stream of passive income ("Bucks") over time.
* **Rarity & Progression:** Animals have different rarities ranging from Common to Devine, and a leveling system scales their stats as you progress.
* **Monetization (Stealing):** A unique mechanic allows players to pay (via Developer Products) to "steal" animals that are placed in other players' bases.

## Architecture

The codebase is structured using a clean **Service/Controller architectural pattern**, typical of modern Roblox development:
* **Custom Framework:** Utilizes a custom `ModuleLoader` to handle dependencies and lifecycle methods for services and controllers.
* **`src/Server/Services`:** Contains the core game logic (e.g., `AnimalService`, `BaseService`, `WeaponService`, `DataService`).
* **`src/Client/Controllers`:** Handles user interfaces, local effects, and player input (e.g., `EffectsController`, `InventoryController`).
* **`src/Shared`:** Contains shared modules like `GameConfig.luau`, which holds all the balancing data, animal stats, and global constants.

## Getting Started

This game template was generated by Argon.

To build the place in the project root use:

```bash
argon build
```

To begin syncing open Roblox Studio and start Argon server using:

```bash
argon serve
```
